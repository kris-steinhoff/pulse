# Pulse

Pulse is a personal terminal-based dashboard application built with Python and Textual.

## Installation

Install the application globally using [`uv`](https://docs.astral.sh/uv/):

```bash
uv tool install git+https://github.com/kris-steinhoff/pulse
```

To get the latest version, use:

```bash
uv tool upgrade pulse
```

## Usage

Run the dashboard:

```bash
pulse
```

## Customization

Everything is hard-coded. Fork this repo and vibe code your own customizations with your favorite AI assistant.
